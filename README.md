## 1. 서비스 소개
> 아이한글은 다문화 가정 아동을 위한 AI 기반 한글 공부 어플리케이션입니다.
> 다문화 가정 아이들 중, 한글 책 학습에 어려움을 겪는 아이들의 자신감과 자아존중감을 향상시키고, 언어가 유창해지도록 도와주기 위해 만들어졌습니다.
